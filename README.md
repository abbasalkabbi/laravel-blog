## Getting Started
1. Clone the repo
   ```sh
   git clone https://github.com/abbasalkabbi/laravel-blog.git
   ```
2. Install NPM packages
   ```sh
   composer install
 
3. Install NPM packages
   ```sh
   npm install
   ```
